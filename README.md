# Stock Trading Platform

## Project Overview
A comprehensive Stock Trading Platform built in Java with a professional Graphical User Interface (GUI). This application simulates a real-time trading environment where users can buy and sell stocks, track their portfolio performance, and monitor market activity.

## Features

### User Authentication
- **Login Screen**: Secure login with username and password
- **Show/Hide Password**: Toggle password visibility with eye icon
- **User Registration**: Create new accounts with validation
- **Multi-User Support**: Multiple users with separate portfolios

### Real-Time Market Simulation
- **Live Market Data**: Stock prices update every 5 seconds
- **Price Movements**: Random price changes between -2% and +2%
- **Stock Categories**: Technology, Consumer, Automotive, Entertainment, Financial, ETF
- **Market Indicators**: Price, change percentage, high/low, volume
- **10+ Stocks**: Pre-loaded with real-world company names

### Trading Operations
- **Buy Stocks**: Purchase shares with real-time pricing
- **Sell Stocks**: Sell shares from your portfolio
- **Trade Confirmation**: Confirm before executing trades
- **Real-time Pricing**: Stock prices fluctuate automatically
- **Balance Checking**: Insufficient funds protection

### Portfolio Management
- **View Holdings**: See all stocks you own
- **Profit/Loss Tracking**: Monitor performance per stock
- **Portfolio Value**: Total portfolio value in real-time
- **Cash Balance**: Track available funds
- **Average Price**: See your average purchase price

### Transaction History
- **Complete Log**: All buy/sell transactions recorded
- **Transaction Details**: Shares, price, total, timestamp
- **Status Tracking**: Completed status for all transactions

### Dashboard
- **Statistics**: Total stocks, active users, transactions, market day
- **Quick Actions**: Refresh data, trade stocks, logout
- **Market Status**: Real-time market day and up/down count

### Data Persistence
- **SQLite Database**: All data stored in a local database
- **Auto-Save**: Data automatically saved after every transaction
- **Auto-Load**: Previous data loaded on startup
- **Tables**: Stocks, users, portfolios, transactions

## Technologies Used
- Java 17
- Swing (GUI Framework)
- SQLite (Database)
- JDBC (Database Connectivity)
- SLF4J (Logging)
- Collections Framework
- Timer (Market Simulation)
